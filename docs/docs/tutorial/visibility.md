# Interface and Menus



## Main Menu?

The kRONOS main menu is accessed through the w key. 

## Pie Menus

Holding W + any of the following keys and moving the mouse will open a unique PIE menu.

D

A 

SPACE

SHIFT

Pies can be configured in the preferences panel.
